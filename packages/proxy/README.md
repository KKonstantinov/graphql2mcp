# graphql-to-mcp

Standalone proxy that converts GraphQL endpoints into MCP (Model Context Protocol) servers. Point it at a GraphQL endpoint and get an MCP server with tools mapped from queries and mutations.

## Installation

```bash
npm install -g graphql-to-mcp
```

## Usage

```bash
graphql-to-mcp --endpoint https://api.example.com/graphql
```
